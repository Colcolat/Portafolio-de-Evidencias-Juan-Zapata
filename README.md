## Contenido del Repositorio

Este portafolio integra diversas calculadoras y visualizadores matemáticos desarrollados con tecnologías web modernas.

| Actividad | Título | Descripción Funcional |
| :---: | :--- | :--- |
| **02** | **Recta Numérica** | Visualización interactiva de conjuntos numéricos (Naturales, Enteros, Racionales, Irracionales). |
| **04** | **Números Complejos** | Calculadora de conversión entre formas Cartesiana, Polar y Exponencial. |
| **06** | **Expresiones Algebraicas** | Traducción de lenguaje natural a expresiones algebraicas con base de datos de patrones. |
| **08** | **Calculadora de Variables** | Definición de variables y resolución de ecuaciones lineales personalizadas. |
| **10** | **Productos Notables** | Expansión de binomios y visualización geométrica de áreas (Cuadrados y Conjugados). |
| **12** | **Factorización** | Algoritmo paso a paso para la división de polinomios, soportando raíces complejas. |
| **14** | **Ecuaciones de 1er grado** | Motor de despeje de ecuaciones lineales y fórmulas físicas/matemáticas. |

## Tecnologías y Características

Este proyecto no utiliza plantillas genéricas. Todo el diseño y la lógica han sido implementados manualmente.

* **Frontend:** HTML5 Semántico, CSS3 (Variables, Grid, Flexbox, Animaciones Keyframes).
* **Lógica:** JavaScript y mucha fé.
* **Librerías Matemáticas:** `Math.js` para el cálculo simbólico y parsing de expresiones complejas.
* **Multimedia:**
    * **Estética Vaporwave:** Paleta de colores Neón (Cian, Rosa, Morado), fondos de rejilla 3D y efectos de brillo (Glow).
    * **Sistema de Audio:** Reproductor flotante con persistencia entre páginas (el audio no se reinicia al navegar).
    * **Diseño Responsivo:** Adaptable a dispositivos móviles y escritorio.
